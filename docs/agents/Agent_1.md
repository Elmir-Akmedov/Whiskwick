# Agent 1 - Character Base Sprites

## Objective
Create the base character sprite pack in Stardew Valley-inspired cozy 2D pixel art for Whiskwick.

## Inputs
- docs/DESIGN_AGENT_PROMPTS.md (sections 1, 2, 3)

## Tasks
1. Create player baker base sprite.
2. Create 4 worker role sprites (baker, cashier, runner, cleaner).
3. Create 6 customer archetype sprites.

## Output Files
- assets/art/generated/characters/player_baker_base.png
- assets/art/generated/characters/workers_set.png
- assets/art/generated/characters/customers_set.png

## Constraints
- Transparent background.
- Readable at 32 to 48 px character height.
- Fixed camera angle and consistent pixel scale.
- No text, logos, or watermark artifacts.

## Acceptance Criteria
- Role identity is clear from silhouette alone.
- All sprites share one palette family and shading style.
- Character sizes are consistent for gameplay.
