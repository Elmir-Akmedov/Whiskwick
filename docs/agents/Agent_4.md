# Agent 4 - Tiles And Materials

## Objective
Create seamless tile sheets for core rooms and transitions.

## Inputs
- docs/DESIGN_AGENT_PROMPTS.md (section 6)

## Tasks
1. Sales room floor tile variants.
2. Kitchen floor tile variants.
3. Warehouse floor tile variants.
4. Transition/threshold and queue marker tiles.

## Output Files
- assets/art/generated/tiles/room_tiles_atlas.png

## Constraints
- 32 px tile target.
- Seamless repetition.
- Low-noise texture patterns for readability.

## Acceptance Criteria
- No visible seams in repeated tiling tests.
- Room identity is clear by tile family.
- Palette aligned with the rest of the asset pack.
